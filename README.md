## Propósito do Repositório

Este repositório reúne um sistema completo para detecção de pulos usando visão computacional e machine learning, com aplicação em jogos interativos.

### Breve explicação dos módulos principais:
- **frames-capture**: Captura imagens da webcam para criar o dataset de pulos e não pulos.
- **image-augmentation**: Aplica técnicas de aumento de dados para expandir o dataset.
- **image-processing**: Redimensiona e padroniza imagens para uso em modelos de machine learning.
- **data**: Scripts e notebooks para preparar e rotular os dados em formato CSV.
- **model**: Notebooks para treinar, validar e avaliar o modelo SVM de detecção de pulos.
- **dino-game**: Jogo interativo controlado por pulos reais detectados pela webcam.

Cada módulo possui documentação detalhada na pasta `docs/`.
